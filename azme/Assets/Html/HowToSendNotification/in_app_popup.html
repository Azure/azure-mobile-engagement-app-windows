Make sure the app is open and then in the Azure Mobile Engagement Portal:

  1. Go to the "Reach" tab, and then select "Announcements".
  2. Click on "New Announcement".
  3. Fill the following values in the form:
    • Name: Provide the name of the campaign to identify it.
    • Category: Provide the category as "POP-UP"
    • Delivery type: "In-app notification"
    • Delivery time: "In session"
    • Title & Message: Provide the title and the message for the popup screen.
    • Content type - Select "Notification only"
    • Action URL - Optionally provide the Action URL as "engagement://demo/product" that corresponds to the native app screen which will open up when the notification is clicked.
  4. Finally, either send the test notification or create and activate the campaign to see the popup notification.
