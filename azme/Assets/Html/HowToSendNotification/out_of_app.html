Make sure the app is open and then in the Azure Mobile Engagement Portal:

  1. Go to the "Reach" tab, and then select "Announcements".
  2. Click on "New Announcement".
  3. Fill the following values in the form:
    • Name: Provide the name of the campaign to identify it.
    • Category: Leave it as blank with no values.
    • Next, choose the type of system notification you want to send:
    • Delivery time: "Any time"
    • Title & Message: Provide the title and the message for the system notification.
    • Type of the announcement:
      • "Notification only": Clicking the notification will open a native screen of the app
      • "Web announcement": Clicking the notification will open a screen of the app that contains a web view
    • Add the action URL "engagement://demo/recent_updates" :
      • For "Notification only" - this will open up when the notification is clicked in the notification center.
      • For "Web announcement" - this will open up when the Action button on the announcement will be clicked.
    • Provide the Title and Announcement content if you chose Web announcement previously.
    • Specify Action button as "OK which will open up the Action URL specified above and Exit button as Cancel.
  4. Finally, either send the test notification or create and activate the campaign to see the notification.
