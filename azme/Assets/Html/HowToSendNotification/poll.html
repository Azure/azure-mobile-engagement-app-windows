Make sure the app is open and then in the Azure Mobile Engagement Portal:

  1. Go to the "Reach" tab, and then select "Polls".
  2. Click on "New poll"
  3. Fill the following values in the form:
    • Name: Provide the name of the campaign to identify it.
    • Category: Leave it as blank with no values.
    • Delivery time: "In session"
    • Title & Message: Provide the title and the message of the initial notification which the user clicks to go to the poll screen.
    • Content Title, Description : Provide the title and the description which will be shown on the poll screen.
    • Action button: Provide a text of "Submit"
    • Exit button: Provide a text of "Cancel"
    • Finally add some poll/survey questions and options for each.
  4. Finally, either send the test notification or create and activate the campaign to see the poll/survey notification.
